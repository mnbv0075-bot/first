# 🎬 영화 리뷰 감성 분석 서비스

영화 제목을 검색하면 실제 리뷰를 자동으로 수집하고,
직접 학습시킨 AI 모델이 **긍정/부정을 판별**해서 종합 평가를 알려주는 웹 서비스입니다.

## 😀 주요 기능

- **리뷰 감성 분석** : 리뷰를 입력하면 긍정/부정 자동 판별
- **영화 통합 검색** : 영화 제목 검색 → TMDB API로 실제 리뷰 수집 → 전체 리뷰 분석
- **종합 판정 시스템**
  - 80% 이상 → 🏆 강력 호평작!
  - 60~80% → 👍 대체로 호평
  - 40~60% → 😐 호불호 갈리는 영화
  - 40% 미만 → 💀 혹평작...
- **한국어 번역** : 영어 리뷰를 한국어로 번역해서 표시
- **포스터/개봉연도 표시** : 검색한 영화의 정보를 함께 제공

---

## 🤖 AI 모델

컴퓨터는 글을 읽지 못하기 때문에, 두 단계로 리뷰를 분석합니다.

1. **TF-IDF** : 리뷰 속 감정이 담긴 중요한 단어에 점수를 매겨 숫자로 변환
2. **로지스틱 회귀** : 긍정 단어와 부정 단어의 무게를 재는 저울처럼 판정

| 항목 | 내용 |
|:---|:---|
| 학습 데이터 | IMDB 영화 리뷰 50,000개 (aclImdb) |
| 벡터화 | TF-IDF (최대 특징 개수 20,000) |
| 모델 | 로지스틱 회귀 |
| **정확도** | **88.0%** |

> 모델 학습에는 **IMDB 리뷰 5만 개**를 사용했고,
> 실제 서비스의 리뷰는 **TMDB API**로 실시간 수집합니다.

---

## 🛠 사용 기술

- **Python** - 전체 개발 언어
- **scikit-learn** - AI 모델 학습 (TF-IDF + 로지스틱 회귀)
- **FastAPI** - 웹 서비스 및 API 구현
- **TMDB API** - 영화 정보 및 리뷰 수집
- **deep-translator** - 영어 리뷰 한국어 번역

---

## 📁 폴더 구조
